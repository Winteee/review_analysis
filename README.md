# review_analysis
### 利用Keras进行评论的情感极性分析
- 此为Winteeena的本科毕业论文选题，通过Keras提供的框架进行中文情感判断训练（也是我用python跑通的第一个程序 (^_^)）
- 数据来源：大众点评-上海美食商户的30000条评论（review30000.csv）

### 实验结果
- 针对单条评论的正负性预测准确率达93%
